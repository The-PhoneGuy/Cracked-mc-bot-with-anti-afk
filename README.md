This project need: node.js,mineflayer,node-schedule

Step 1:
Download bot.js

Step 2:
Create a new directory

Step 3:
Install dependencies in directory:
npm install mineflayer node-schedule

Step 4:
Replace 'username' , 'server' and 'port' by your informations

Step 5:
Change logout time

Step 6:
Run bot.js with:
node bot.js
